# 🌙 Moonfruit Tap

这是一个个人维护的 Homebrew Tap 仓库，包含了一些官方仓库中不包含的软件包和字体。

## 📦 安装方法

### 添加 Tap

```bash
brew tap moonfruit/tap
```

### 安装软件

```bash
brew install moonfruit/tap/<formula>
# 或
brew install --cask moonfruit/tap/<cask>
```
